# MycoFlow IEEE Poster

## Files

- **new-poster.html** — Production-ready IEEE poster (A1 Portrait, 1188×1680px @ 150 DPI)
- **poster.html** — Legacy version (large, not recommended)
- **MycoFlow_Poster_A1.pdf** — Pre-rendered PDF version
- **translate.py** — Translation utility for multi-language support

## Features

✅ **IEEE Submission Ready**
- A1 Portrait (594×841 mm)
- Print-optimized CSS (@media print)
- 150 DPI native resolution
- Professional color profile (sRGB for web, suitable for CMYK print)

✅ **Export Options**
- 🖨 **Print to PDF** — Direct browser print (Ctrl+P or Print button)
- 💾 **Download PNG** — High-res screenshot (1188×1680px)

✅ **Accessibility**
- Semantic HTML5
- ARIA labels on main content
- Descriptive alt text for QR codes
- Color contrast meets WCAG AA

✅ **Technical**
- Google Fonts with fallbacks
- Responsive scaling for any viewport
- QR codes for: Portfolio | MycoFlow | GitHub
- External dependency: html2canvas (for PNG export)

## How to Use

1. **View in Browser**: Open `new-poster.html` in any modern browser
2. **Print to PDF**: Click "Print to PDF" button or Ctrl+P → Save as PDF
3. **Download PNG**: Click "Download PNG" button (requires internet for html2canvas)
4. **Print Locally**: Use the browser's print dialog for best results

## Print Settings

For optimal printing:
- **Paper**: A1 (841 × 1189 mm)
- **Orientation**: Portrait
- **Scale**: 100% (no scaling)
- **Margins**: 0mm (if available)
- **Color Mode**: RGB or CMYK (printer will handle conversion)
